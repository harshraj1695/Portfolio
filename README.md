## 🛠 Run Locally

Clone the repository:

``` bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
```

Install dependencies:

``` bash
npm install
```

Start development server:

``` bash
npm run dev
```

Build for production:

``` bash
npm run build
```
